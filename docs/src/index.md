# Bagyo.jl

Bagyo.jl is a package for extraction and analysis of typhoon and weather data affecting the Philippine Area of Responsibility (PAR).

## Installation
To install the package, run the following:
```julia
julia> using Pkg
julia> Pkg.add("Bagyo")
```
